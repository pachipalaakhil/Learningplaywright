# Learning Playwright

Small repo for Playwright experiments and examples.

## Structure

- Chapter 1: basic examples
- Chapter 2: (empty)

## Quick start

Install dependencies (if you plan to use Playwright tests):

```powershell
npm init -y
npm i -D @playwright/test
npx playwright install
```

Run Playwright tests:

```powershell
npx playwright test
```

Run simple Node example:

```powershell
node "Chapter 1/HelloProgram.js"
```
